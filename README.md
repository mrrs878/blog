## 部署/运行

### 手动

1. `git clone git@github.com:mrrs878/blog.git`

2. `yarn install && yarn start`

## Docker一键部署

1. 安装Docker

2. 安装docker compose

新建`blog`目录，复制`docker-compose.yml`至`blog`下，运行`docker-compose up -d`
